# Docker image for RNA-Seq processes

It is based on `ubuntu-17.04` version of [`docker.io/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included bioinformatics tools:
------------------------------
* affy (v1.56.0)
* Cufflinks (v2.2.1)
* Cutadapt (v1.14)
* DESeq2 (v1.16.1)
* FastQC (v0.11.5)
* HISAT2 (v2.1.0)
* HTSeq (v0.9.1)
* RSEM (v1.4.0)
* rtracklayer (v1.38.0)
* SAMtools (v1.5)
* STAR (v2.5.3a)
* tximport (v1.6.0)
