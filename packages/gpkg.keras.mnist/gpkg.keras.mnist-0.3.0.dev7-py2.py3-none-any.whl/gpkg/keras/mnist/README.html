Guild AI - Keras MNIST package

The keras.mnist package contains various MNIST models that are provided in https://github.com/keras-team/keras/tree/master/examples.

Refer to guild.yml for details.

To install this package, run:

$ guild install keras.mnist

To list available models, run:

$ guild models keras.mnist

To train a model, use:

$ guild train MODEL [epochs=NUM] [batch-size=NUM]
