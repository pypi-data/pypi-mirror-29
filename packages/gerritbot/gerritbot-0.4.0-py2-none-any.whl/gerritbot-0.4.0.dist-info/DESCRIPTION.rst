=========
GerritBot
=========

Gerritbot is an IRC bot that will notify IRC channels of Gerrit events.

To install::

    $ sudo python setup.py install

Online documentation:

 * http://docs.openstack.org/infra/system-config/gerrit.html#gerrit-irc-bot
