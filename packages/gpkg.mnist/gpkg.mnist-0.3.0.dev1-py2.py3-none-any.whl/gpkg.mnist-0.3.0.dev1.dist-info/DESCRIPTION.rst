
Models
######

mnist-softmax
#############

*Softmax regression classifier for MNIST*

Operations
==========

train
^^^^^

*Train the softmax regression*

Flags
-----

**batch-size**
  *Number of images to include in a training batch (default is 100)*

**epochs**
  *Number of epochs to train (default is 10)*
